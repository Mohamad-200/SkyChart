# SkyChart

SkyChart is a JavaScript charting library focused on simple, expressive 2D and 3D charts for modern websites.

## Status

This package folder is the public distribution workspace.

- Source code is being prepared incrementally.
- `SkyChart.js` is intentionally not included yet.
- Licensing is dual-license (community + commercial).

## Planned Package Layout

```text
skychart/
  src/
  dist/
  examples/
  LICENSE-SKYCHART.md
  CHANGELOG.md
  package.json
```

## Development

Initial commands (after implementation files are added):

```bash
npm install
npm run build
npm run test
```
